# ScootShare

## Description
This repository contains the code for building and deploying the ScootShare application.

ScootShare allows for users to share motorcycles of different types, such as, Vespas. 

The backend architecture uses S3 to host the static website. Authentication is handled through AWS Cognito, while AWS Lambda and AWS API Gateway handle HTTP REST operations.

AWS DynamoDB is used to store the data, whereas S3 serves to store photos and images. AWS Cloudfront is used as a CDN to display static resources.
